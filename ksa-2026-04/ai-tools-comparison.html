ChatGPT (OpenAI) 과정 기본 도구
무료: GPT-5.3 · 광고포함 Go ₩11,000/월 · 광고 Plus ₩29,000/월 Pro(5x) ₩159,000/월 Pro(20x) ₩299,000/월
강점: 파일 업로드 후 Python 코드 실행(Advanced Data Analysis)으로 기술통계·교차분석·시각화 원스톱 처리. Deep Research(Plus 10회/월). 가장 넓은 사용자 기반 → 수강생 친숙도 최고. 긴 대화 맥락 유지 우수.
약점: 무료/Go 티어 2026.2~ 광고 포함. 수치 환각 가능성 → 반드시 Excel 교차검증. 분석 결과를 Excel로 직접 반영 불가(별도 다운로드 후 수동 이동).
Claude (Anthropic) 해석·보고서 특화
무료: Sonnet · 20~30msg/일 Pro ₩29,000/월 · 5배 용량 Max(5x) ₩145,000/월 Max(20x) ₩290,000/월 Team $25~30/석
강점: 긴 문서 분석 최강(최대 100만 토큰). 인사이트 언어화·보고서 초안 품질 최우수 — "해석→판단 연결" 서술 특화. Artifacts로 코드·문서 실시간 생성. Projects로 과정 자료 묶어 관리. 광고 없음. 유료 시 학습 데이터 미사용.
약점: 데이터 분석 UI 직관성은 ChatGPT 대비 약간 낮음. 이미지 생성 불가. 무료 일일 메시지 한도 소진 빠름.
Claude in Excel Pro ₩29,000/월에 포함
무료: 없음 Pro 구독에 포함 · 추가 비용 없음
강점: Excel 사이드바에서 워크북 직접 분석 — 수식 추적, 피벗테이블 편집, 차트 수정, 조건부 서식. 셀 레벨 인용으로 검증 투명성. 가정 변경 시 수식 의존성 보존 + 영향도 분석. Excel↔PowerPoint 대화 컨텍스트 공유. Sonnet/Opus 모델 선택 가능.
약점: VBA·Power Query·데이터 테이블 미지원. .xlsx/.xlsm만 지원. 인터넷 필수. Claude 전체 사용량에서 차감(별도 할당 아님).
Google Gemini 무료 티어 최고
무료: Flash 모델 · 관대한 한도 Pro ₩29,000/월 · 2TB 포함 Ultra ~₩180,000/3개월
강점: 무료 티어 업계 최고(이미지 생성·코드 실행·웹 검색 포함). Google Workspace 네이티브 통합(Sheets·Docs·Slides). Deep Research 우수. 구글 생태계 사용자에게 진입장벽 최저.
약점: Excel/Power BI 워크플로우와 직접 통합 안 됨 — 본 과정 도구 체인과 거리. 세밀한 수치 정확도에서 약간 열세. Sheets 전용이라 .xlsx 직접 처리 제한적.
Microsoft Copilot (365) M365 + 추가 과금
Chat: M365 구독자 무료 Pro $20/월 (개인) Business $18~30/석/월 + M365 실질 $27~60/월
강점: Excel·Word·PPT·Teams 완전 네이티브 통합 — 수식 생성, VBA 지원, 슬라이드 작성까지 앱 내 완결. 기업 보안·컴플라이언스 최강. 조직 데이터(SharePoint, Teams) 연결.
약점: M365 구독 전제 → 실질 비용 높음. 분석 추론 깊이 ChatGPT·Claude 대비 열세. 개인 수강생 대상 라이선스 확보 어려움. 한국어 자연어 분석 품질 불안정.
1. 분석 방법 선택 (AI에게 묻기)
ChatGPT Plus ≥ Claude Pro
"매출 하락 원인 파악에 어떤 분석이 적절할까?" — 두 도구 모두 우수. ChatGPT는 즉시 코드 실행까지 연결, Claude는 판단 기준 구조화에 강점.
2. 데이터 전처리 (Power Query 대안)
ChatGPT Plus > Claude in Excel
ChatGPT: 파일 업로드 → Python으로 결측치 처리·피벗 변환 즉시 실행. Claude in Excel: Power Query 미지원이나 간단한 정리·필터는 가능.
3. 기술통계·교차분석 실행
Claude in Excel ≈ ChatGPT Plus
Claude in Excel: 워크북 안에서 직접 피벗·기술통계 → 셀 인용으로 검증 용이. ChatGPT: Python으로 더 복잡한 통계 가능하나 Excel 이동 과정 필요.
4. 시각화·대시보드
Power BI > ChatGPT > Claude in Excel
Power BI는 대체 불가. AI 보조로는 ChatGPT가 matplotlib/plotly 다양한 차트 생성. Claude in Excel은 기본 차트 편집 가능하나 복잡한 대시보드는 한계.
5. 인사이트 언어화·보고서 초안
Claude Pro > ChatGPT Plus
"그래서 뭘 해야 하는가"를 기획서 문장으로 전환하는 품질에서 Claude 우세. 긴 맥락 유지, 과대해석 방지 표현, 판단 근거 구조화에 강점.
6. 텍스트 분석 (VOC·설문)
ChatGPT Plus ≥ Claude Pro
ChatGPT: 라벨링·감성분석·주제 분류를 코드 실행으로 자동화. Claude: 분류 품질 우수하나 대량 처리 시 메시지 한도 소진. 교안에서도 ChatGPT 라벨링 + Excel 집계를 Plan B로 명시.
7. 레드팀 검증·조언 얻기
Claude Pro ≈ ChatGPT Plus
두 도구 모두 가능. Claude는 비판적 분석·반대 입장 구성에서 미묘하게 우세. "이 결론에 반대하는 입장에서 공격해줘" 프롬프트에 더 날카로운 응답.
8. 캡스톤 (기획메모 + AI 로그)
ChatGPT + Claude 병행 추천
분석 실행은 ChatGPT(코드 실행력), 해석·보고서는 Claude(서술 품질). 두 도구 대화 로그를 모두 제출 → AI 협업 역량 평가에서 차별화.

약점: 데이터 분석 자체 불가 — 시각화/보고 단계에만 유용. PPT/Power BI 워크플로우와 중복 가능.

핵심 원칙: AI는 가속화, 판단은 사람
분석 기법을 AI가 대신해도, 질문 설계·결과 검증·서술 설득은 기획자의 영역입니다.
AI 첫 응답 ≠ 최종 결론. AI가 써준 보고서는 보고서가 아닙니다. 여러분이 분석하고, AI가 도와주고, 여러분이 판단한 보고서가 보고서입니다.
도구를 배우는 이유: '스스로 하기 위해'에서 'AI를 제대로 쓰기 위해'로. 원리를 알아야 AI의 오류를 잡을 수 있습니다.
